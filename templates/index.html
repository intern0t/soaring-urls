Usage

- Using the form below

- Using curl

curl --data "url=SOME_LONG_URL_HERE" {{ deploy_url }}

- Create an alias (Linux | Optional)

echo 'alias sim="curl --data-urlencode data@- https://storedin.herokuapp.com/"' >> ~/.bash_aliases && source ~/.bash_aliases
cat somefile.ext | sim

- Prettify (Linux | Optional)

StoredIn responds with JSON formatted data. You can utilize a package called jq to prettify the JSON response. You can pipe the output to jq by appending | jq to the commands above. 

cat somefile.ext | curl --data-urlencode data@- https://storedin.herokuapp.com/ | jq

Features

  • Accepts POST requests with 'data' as the field. (works with any systems without having to install any special tools).

Restriction(s)

  • No illegal contents.
  • File upload is NOT allowed. Raw contents however, is allowed.
